# 🚗 Ykaa Scrap

Simple Scrap Vehicle Script

## 🔧 Installation
- Drag a script file
- Drop into your server resources folder
- Ensure it in server.cfg

## ⚙️ How to use
 - Drive a car to the scrapyard.
 - Use the interaction (E) on the vehicle.
 - Wait for the progress bar to finish.
-  Receive scrap items in your inventory.

## 📋 Depencies
- es_extended
- ox_lib
